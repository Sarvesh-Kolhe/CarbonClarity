# CarbonClarity 🌍

**Understand your carbon footprint. Know what actually matters. Take action without the overwhelm.**

CarbonClarity helps individuals measure, understand, and reduce their personal carbon footprint through a fast onboarding estimate, a clear category-based dashboard, and personalized recommendations ranked by impact-to-effort ratio — so you always know the *next best action*, not just a long list of generic eco-tips.

## Why CarbonClarity?

Most carbon-tracking apps either demand tedious manual logging or bury users in vague advice ("use less plastic!") that barely moves the needle. CarbonClarity focuses on what actually works:

- **📊 Fast, accurate baseline** — Get your estimated footprint in under 3 minutes using home, transport, diet, and consumption data — refined over time with real bills and transaction data.
- **🎯 Prioritized, personalized actions** — Recommendations ranked by real CO₂e impact vs. effort, so you tackle the highest-leverage changes first instead of low-impact busywork.
- **🔄 Low-friction tracking** — Passive tracking (location, utility/bill uploads) and one-tap logging instead of manual diaries.
- **📈 Clear, honest insights** — See your footprint by category (Home, Transport, Food, Stuff), benchmarked against national averages and science-based targets.
- **🏆 Sustainable habit-building** — Streaks, milestones, and progress framing designed to motivate without guilt or eco-shame.

## Features

- Smart onboarding & footprint estimator (EPA/DEFRA-based emission factors)
- Category-level dashboard (Home / Transport / Food / Stuff)
- Personalized action recommendation engine (impact ÷ effort scoring)
- Passive + one-tap tracking
- Progress streaks, milestones, and monthly recap insights
- Built for extensibility — bring your own emission-factor datasets or integrations

## Tech Stack

- **Frontend:** React Native
- **Backend:** Node.js / Python + PostgreSQL
- **Emission data:** EPA, DEFRA, Climatiq API
- **Integrations:** Plaid (transactions), utility bill OCR
